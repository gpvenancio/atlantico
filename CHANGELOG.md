# Changelog

All notable changes to the "atlantico" extension will be documented in this file.

This project follows the spirit of [Keep a Changelog](https://keepachangelog.com/), with concise notes focused on user-facing changes.

## [Unreleased]

## [1.1.9]

### Changed

- Added Open VSX Registry installation information.
- Updated README positioning for Visual Studio Code and compatible editors.
- Added Open VSX links to the project documentation.

## [1.1.8]

### Changed

- Updated the extension description to reflect availability for Visual Studio Code and compatible editors.

## [1.1.7]

### Changed

- Updated the extension icon.

## [1.1.6]

### Changed

- Added the Atlantico project page as the extension homepage.
- Updated extension metadata to point to the dedicated project page.

## [1.1.0]

### Added

- Expanded language support and visual tuning across multiple languages.
- Added screenshots for JavaScript, TypeScript, Python, Rust, SQL, SCSS, HTML, and the integrated terminal.
- Added a documented color palette for UI, syntax, and terminal colors.

### Changed

- Improved README structure and project presentation.
- Refined theme positioning around calm, low-noise focused development.

## [1.0.0]

### Added

- Initial release of Atlantico.
- Added the core dark theme palette.
- Added semantic highlighting rules.
- Added integrated terminal ANSI colors.
- Added support for common web, scripting, systems, data, and configuration languages.
