# Atlantico

A premium low-noise dark theme for Visual Studio Code.

## Features

• Soft cinematic palette
• Minimal visual noise
• Comfortable long-session coding
• Multi-language support
• Refined UI consistency

## Supported Languages

JavaScript, TypeScript, React, Python, Go, Rust, C/C++, Java, HTML, CSS, JSON, Markdown and more.

## Development

The theme design, palette tuning and token refinement were iterated with the help of AI assisted tooling.
